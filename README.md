# Human vs Monkey Image Classification

This is a simple image classification project for beginners.

The project classifies images into two categories:
- Human
- Monkey

---

## Project Structure

human-vs-monkey/
- dataset/
  - human/
  - monkey/
- image_classifier.py
- requirements.txt
- README.md

---

## Technologies Used

- Python
- TensorFlow
- CNN (Convolutional Neural Network)

---

## How to Run

1. Install TensorFlow  
   pip install tensorflow

2. Run the program  
   python image_classifier.py

---

## Dataset

The dataset contains images of humans and monkeys.
Images are placed inside the dataset folder.
